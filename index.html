Convo uses Google Sheets as its database via Google Apps Script. Follow the steps below to get started.

Setup Instructions

1

Create a new Google Sheet and note its ID from the URL:
https://docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit

2

Open Extensions → Apps Script in your Sheet. Paste the Apps Script code (click the button below), then click Deploy → New deployment → Web App. Set "Who has access" to Anyone.

3

Copy the generated Web App URL and paste it below.

4

Add these sheet tabs in your Google Sheet: Users Messages
