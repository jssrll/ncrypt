Connect to Google Sheets

This app uses Google Sheets as its database. Set up your Apps Script backend in a few steps, then paste the Web App URL below to get started.

Quick Setup Guide

1

Open Google Sheets, create a new spreadsheet, then go to Extensions → Apps Script.

2

Replace the default code with the contents of Code.gs (provided alongside this file). Save with Ctrl+S.

3

Click Deploy → New deployment.
Type: Web app  ·  Execute as: Me  ·  Who has access: Anyone.
Click Deploy and authorize when prompted.

4

Copy the generated Web App URL and paste it below. The Users sheet will be created automatically on first use.
